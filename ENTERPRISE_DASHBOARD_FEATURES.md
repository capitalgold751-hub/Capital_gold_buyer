# Capital Gold Enterprise Dashboard

Included in this build:
- MongoDB authentication and sessions
- Admin/staff access
- Leads CRM and appointment assignment
- GoldAPI sync, manual override, fallback and rate history
- Separate admin Settings page
- Email/WhatsApp alert enablement, recipients and test actions
- Indicative buying percentage control
- Stale-rate threshold and alert cooldown
- Staff account creation, password reset and enable/disable
- Branch and Blog CMS
- CSV reports for leads, appointments and rates
- Administrator audit log
- Security and integration-health views

Secrets remain in `.env.local` or Vercel environment variables.
